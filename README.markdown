Introduction
============
Puppy is a (driver) meta-framework for Python Web developers.

Requirements:
-------------
* Linux
* [pyinotify][pyinotify]
* python2.5/python2.6
* [subcommand][subcommand]
* [PyYAML][pyyaml]

[pyyaml]: http://pyyaml.org/
